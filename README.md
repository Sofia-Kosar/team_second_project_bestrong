## Git guide

## Sync `main`

```bash
git checkout main
git fetch origin
git pull origin main
```

## Create your branch and push it

Replace `[YOUR_BRANCH]` with your branch name.

```bash
git checkout -b [YOUR_BRANCH]
git push [YOUR_BRANCH]
```

## Open a pull request

Create a pull request from `[YOUR_BRANCH]` to `main`.
